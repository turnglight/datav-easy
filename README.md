# datav-easy
|  插件   | 说明  |
|  ----  | ----  |
| vue  | vue |
| vue-router  | 路由 |
| vuex  | 状态机 |
| eslint  | 代码格式检查插件 |
| element-ui  | ui组件 |
| axis  | http请求插件 |
| echarts | 图形插件|
| echarts-easy-* | echarts经过二次封装的图形组件 |
| html2canvas|保存画布，对画布进行截屏|
| vue-drag-resizeable| canvas拉拽缩放功插件 |
| vue-ruler-tool| 标尺辅助线插件，不稳定被弃用|
| webpack| 项目打包插件 |

## 安装
```
yarn install
```

### 热启动
```
yarn serve
```

### 编译
```
yarn build
```

### lint文件格式修复
```
yarn lint
```
